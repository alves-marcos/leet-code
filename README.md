# LeetCode Solutions

Repositório com minhas soluções para problemas do LeetCode desenvolvidas em **Java**.

O objetivo é praticar lógica de programação, estruturas de dados e algoritmos, registrando minha evolução na resolução de desafios.

## Tecnologias

* Java
* Git
* GitHub
* IntelliJ

## Estrutura

Cada problema possui sua própria pasta contendo a solução e, quando necessário, uma breve explicação da abordagem utilizada.

---

Este repositório é atualizado continuamente conforme novos problemas são resolvidos.
